# ANOR TECH — Corporate Website

Одностраничный корпоративный сайт компании ANOR TECH.

## Технологии
- Чистый HTML5 / CSS3 / Vanilla JS
- Без фреймворков и зависимостей — один файл `index.html`
- Мультиязычность: RU / EN / UZ (переключатель в шапке)
- Формы: интеграция с [Formspree](https://formspree.io)

## Структура
```
anortech-site/
└── index.html       # Весь сайт в одном файле
└── README.md
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub (например `anortech-site`)
2. Загрузите файлы
3. Перейдите в **Settings → Pages**
4. Source: **Deploy from a branch** → ветка `main` → папка `/ (root)`
5. Нажмите **Save**

Сайт будет доступен по адресу:
`https://<ваш-username>.github.io/anortech-site/`
